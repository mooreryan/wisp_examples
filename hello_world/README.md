# Hello, World!

A classic Hello, World!

Shows some basic Wisp things like:

- Setting up the handler
- Starting the server
- Handling requests
- Basic routing
- Middleware
- Using [lustre](https://github.com/lustre-labs/lustre) for HTML templating

Gleam dependencies:

```
gleam add mist wisp gleam_erlang lustre
```
